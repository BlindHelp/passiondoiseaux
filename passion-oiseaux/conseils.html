Conseils de navigation avec la revue d'écran Jaws / NVDA

Navigation à travers des éléments du sommaire Passion d'oiseaux

Ces conseils s'adressent tout particulièrement aux utilisateurs de Jaws.
Depuis le sommaire Passion d'oiseaux, vous pourrez atteindre plusieurs dizaines de pages de descriptions relatives à des oiseaux que je connais bien. Pour ouvrir une page de description, il vous suffit de presser la touche entrée sur le nom d'un oiseau , chacun de ces noms étant bien évidemment un lien hypertexte.
A l'intérieur de chacune des pages, plusieurs chapitres vous seront proposés et il sera possible de les atteindre par l'intermédiaire de la touche "T" pour chapitre suivant, et "Maj. +T" pour chapitre précédent. Cependant, je rappelle que Jaws met a votre disposition la liste des différents titres de votre page, par l'intermédiaire du raccourcit clavier "insère + f6".
Pour les utilisateurs de NVDA ces chapitres il sera possible de les atteindre par l'intermédiaire de la touche "H" pour chapitre suivant, et "Maj. +H" pour chapitre précédent. Cependant, je rappelle ainsi que pour Jaws, NVDA met aussi a votre disposition la liste des différents titres de votre page, par l'intermédiaire du raccourcit clavier "NVDA + f7".
Les liens "Retour au sommaire Passion d'oiseaux" sont situés en haut et en bas de toutes les pages.

Activer le son pour chaque page

Au début de chaque page, vous trouverez le bouton Lire pour lancer le son attribué à cette page.
Lorsque vous consultez certaines pages, vous aurez le son associé à la page (par exemple, cela peut être un son de bruitage, Dans les pages avec le nom de l'oiseau, vous aurez le son de l'oiseau associé)
Dans le cas du son des oiseaux dans chaque page html, j'ai précédé la phrase:
Son cri :
avant le bouton "Lyre"
Dans le cas de l'effet sonore de la page dans chaque page html Où il y a un bruitage, j'ai précédé la phrase:
Effet sonore de la page :
avant le bouton "Lyre"
Faire barre d'espace pour lire ou mettre en pause le dite son.
Depuis le sommaire Passion d'oiseaux vous trouverez le lien "Ici quelques chants a la demande"
il vous suffit de presser la touche entrée sur Ce lien.

Une fois cette page ouverte, vous trouverez le premier titre comme nom :.
titre niveau 1 Chants a la demande.
Sous ce titre Vous trouverez les instructions pour son utilisation.

Mais ici, je mets d'autres instructions Un peu plus étendu si vous utilisez un lecteur d'écran lors de la navigation sur la liste déroulante contenant les Chants a la demande :.

Pour les personnes qui ont le lecteur d'écran NVDA vous devez passer en mode formulaire en faisant:
NVDA+Espace, activer et désactiver le mode formulaire automatique; Si vous avez sélectionné le mode formulaire manuel, voici 2 commandes à connaître:
Entrée, Activer le champ de formulaire;
Échappement, désactiver le champ de formulaire.
Pour les personnes qui ont le lecteur d'écran Jaws vous devez passer en mode formulaire en faisant:
+ du pavé numérique ou Entrée. Pour quitter le mode Formulaire, appuyez sur + du pavé numérique ou ÉCHAP.
Dans les deux cas de figure une fois activée la liste déroulante il vous permet de sélectionner un des chants de l'oiseau figurant dans cette liste à l'aide des touches fléchées alors il suffit d'appuyer sur Entrée pour lancer le son de l'oiseau.
Ceci m'a aisément permis de passer en revue tous les sons des oiseaux répertoriés disponibles, juste en me servant d'une main.
Maintenant, plus forcément besoin de se rendre au bouton Aller pour lancer le son de l'oiseau.
Dans les deux cas de figure une fois désactiver le mode formulaire après le titre de niveau 3 appelé :
Lecteur
vous trouverez trois boutons pour:
Mettre play/pause;
Augmenter le volume;
Diminuer le volume.

titre niveau 3 Lecteur
